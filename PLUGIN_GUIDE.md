## 如何使用

### 1. 启动应用
```bash
python main.py
```

### 2. 查看插件
1. 启动后，点击左侧导航栏的 **"插件中心"**（图标：扩展/插件）
2. 您将看到双视图界面：
   - **"我的插件"** - 显示已启用的插件
   - **"插件管理"** - 显示所有插件（已启用和已禁用）
3. 使用顶部的切换按钮在两个视图之间切换
4. 每个插件卡片显示：
   - 插件名称和版本
   - 插件描述
   - 启用/禁用开关
   - 操作按钮（详情、设置）

### 3. 管理插件
- **切换视图**：点击顶部的"我的插件"或"插件管理"按钮
- **启用/禁用插件**：在"插件管理"视图中点击插件卡片上的开关
- **查看详情**：点击信息图标（ℹ️）
- **插件设置**：点击设置图标（⚙️）

### 4. 插件状态持久化
- 插件的启用/禁用状态会自动保存到 `cli_config.json`
- 下次启动应用时会自动恢复插件状态

## 故障排除

### 问题：插件中心显示"暂无可用插件"

**解决方案：**
1. 检查 `plugins/` 目录是否存在
2. 确认每个插件目录下都有 `manifest.json` 文件
3. 查看启动日志中的插件扫描信息

### 问题：插件无法启用

**解决方案：**
1. 检查 `cli_config.json` 中是否有错误的配置
2. 删除 `cli_config.json` 中的 `plugins.disabled_plugins` 数组
3. 重启应用

### 问题：插件功能异常

**解决方案：**
1. 禁用后重新启用插件
2. 查看控制台日志中的错误信息
3. 检查插件的依赖是否满足
