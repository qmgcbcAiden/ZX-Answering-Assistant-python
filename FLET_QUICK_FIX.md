# Flet 可执行文件快速安装

## 🚨 如果程序提示 Flet 可执行文件缺失

### 📥 快速解决（3步）

**第1步：下载**

| 平台 | 下载链接 |
|------|---------|
| Windows | [下载 flet.exe](https://github.com/flet-dev/flet/releases/latest/download/flet-latest-windows-x64.zip) |
| Linux | [下载 flet](https://github.com/flet-dev/flet/releases/latest/download/flet-latest-linux-x64.tar.gz) |
| macOS | [下载 flet](https://github.com/flet-dev/flet/releases/latest/download/flet-latest-macos-x64.tar.gz) |

**第2步：放置**

- **Windows**: 将 `flet.exe` 复制到 `C:\Users\YourName\.flet\bin\`
- **Linux/Mac**: 将 `flet` 复制到 `~/.flet/bin/`

**第3步：完成**

```bash
python main.py
```
